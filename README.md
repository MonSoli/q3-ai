## Возможности

**Чат с ИИ**
- Потоковая генерация ответов в реальном времени
- Режим размышлений (Thinking Mode) с расширенной логикой рассуждений
- Экспорт чатов в PDF

**База знаний**
- Иерархическая структура папок
- Загрузка документов: PDF, DOCX, TXT, изображения и другие форматы
- Drag & drop для загрузки файлов
- Автоматическая сортировка документов по категориям (договоры, акты, счета, письма и т.д.)
- ИИ-агент умеет управлять базой данных
- Полнотекстовый поиск по сообщениям чатов

**Аналитика документов**
- Автоматическая классификация типа документа
- Извлечение сущностей: даты, суммы, организации, ИНН, телефоны, email
- Генерация резюме документов

**Управление пользователями**
- Мультипользовательская система с ролями (админ / пользователь)
- Авторизация по email с JWT-токенами
- Панель администратора: создание пользователей, управление доступом
- Резервное копирование и восстановление базы данных

**Интерфейс**
- Тёмная и светлая тема
- Двуязычный интерфейс (русский / английский)
- Горячие клавиши (Alt+N — новый чат, Ctrl+F — поиск, Escape — закрыть)
- Рендеринг Markdown, таблиц, формул (KaTeX) и подсветка кода

## Требования

- [Python 3.10+](https://www.python.org/)
- [Node.js 18+](https://nodejs.org/)
- [Ollama](https://ollama.com/)

## Быстрый старт

### 1. Запуск Ollama

```bash
ollama serve
ollama pull qwen3:1.7b
```

### 2. Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 3. Frontend

```bash
cd frontend
npm install
npm start
```

Приложение будет доступно по адресу: **http://localhost:3000**
